## 👋 Hey, I'm Waqas!

I'm a **Principal Software Engineer** based in Pakistan with over **7 years** of hands-on experience turning ideas into products people actually use. I've been part of teams that built everything from scrappy startups to multi-tenant SaaS platforms serving thousands of users — and I've loved every bit of the journey.

My sweet spot is the **full stack** — equally comfortable designing a database schema, architecting a RESTful API, or polishing a React UI. Lately, I've been going deep on **Advanced System Design Architecture** and **AWS cloud infrastructure**, because building things that *scale* is what gets me excited.

I write clean, maintainable code and believe that great software is as much about good decisions as it is about good syntax. I care about performance, developer experience, and shipping things that last.

**Right now, I'm open to senior or principal engineering roles** where I can take ownership, contribute to architectural decisions, and work alongside people who care about building things the right way.

- 🔭 Currently mastering **Advanced System Design Architecture & AWS**
- 💬 Talk to me about **PHP, Laravel, Node.js, React, Vue.js, REST APIs, System Design**
- 📫 Drop me a line at **waqasaslam2808@gmail.com**
